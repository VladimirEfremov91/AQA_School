Так вот - сегодня мы наносим ответный удар и сами отнимаем работу у ИИ и пишем свой идеальный TODO-list-app. На этот раз делаем это с применением Git, читаем задание до конца.

Создайте небольшое Java-приложение Task Tracker. Приложение должно состоять минимум из трех классов:

- Task
- TaskTracker
- Main

1. Класс Task описывает задачу. У каждой задачи должны быть характеристики - название задачи и статус (сделана/не сделана) А так же возможность совершить действия:

- пометить задачу как выполненную

- получить статус - выполнена задача или нет

- получить название задачи

- вывести информацию о задаче. пример вывода для невыполненной задачи: [ ] Проверить авторизацию пример вывода для выполненной задачи: [x] Проверить регистрацию

2. Класс TaskTracker хранит список задач.

Внутри используйте структуру список для хранения объектов задач. Добавьте классу возможность совершать следующие действия:

- добавить новую задачу в список

- вывести все задачи

- пометить задачу как сделанную (на базе переданного имени задачи)

- получить статус задачи по переданному названию (выполнена или нет)

- вывести статистику по задачам. вывод в формате: Всего задач: 3 Выполнено: 1 Открыто: 2

3. В классе Main создайте объект TaskTracker и проверьте работу всех его методов.
