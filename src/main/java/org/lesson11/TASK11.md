Исключительная кофе-машина


Создайте 3 класса:
- App
- CoffeeMachine
- NotEnoughWaterException


1.Класс NotEnoughWaterException

Создайте собственное исключение, унаследовав его от RuntimeException. В конструкторе исключения должен приниматься параметр сообщения об ошибке (строка) и передаваться в конструктор родительского класса (через super)

2.Класс CoffeeMachine

В классе CoffeeMachine создайте 3 метода.

2.1 Метод makeCoffee - метод приготовления кофе. принимает целое число - количество миллилитров имеющейся воды - и проверяет хватает ли воды для приготовления кофе (должно быть больше 200). если не хватает - выбрасывает исключение NotEnoughWaterException с подробным сообщением о причине ошибки. если хватает - выводит сообщение о том что кофе приготовлен.

2.2 Метод calculateCups - считает, сколько чашек кофе можно приготовить. принимает в себя аргументы - количество миллилитров воды и объем чашки в миллилитрах.Специально не проверяйте размер чашки на 0. Если размер чашки равен 0, должна возникнуть ошибка ArithmeticException.

2.3 Метод printCoffeeName Метод принимает строку - название кофе. печатает в консоль название кофе большими буквами. Если coffeeName равен null, должна возникнуть ошибка NullPointerException.



3.Класс App

В методе main создать объект CoffeeMachine. Выполнить следующие действия:

3.1. Проверить InputMismatchException

Попросите пользователя ввести количество воды (int). Оберните код в try-catch. Если пользователь ввёл текст вместо числа, выведите: Ошибка: нужно было ввести число

3.2. Проверить своё исключение Вызовите метод: makeCoffee(100); Такого количества воды недостаточно - поймайте NotEnoughWaterException и выведите сообщение из объекта исключения в консоль с помощью System.out.println. В блоке finally выведите строку "Проверка кофемашины завершена"

3.3. Проверить ArithmeticException

Вызовите метод: calculateCups(1000, 0);

Так возникнет деление на 0. Поймайте ArithmeticException и выведите: Ошибка: размер чашки не может быть 0

3.4. Проверить NullPointerException Создайте строковую переменную со значением null. Вызовите метод printCoffeeName передав в качестве аргумента эту переменную.

Поймайте возникшее исключение и выведите: Ошибка: название кофе отсутствует
