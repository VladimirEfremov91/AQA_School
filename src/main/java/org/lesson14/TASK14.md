Задание 14.1 Кинцо

Создайте класс MovieRatingComparator, который реализует интерфейс: Comparator<Movie>

Реализуйте метод compare() так, чтобы фильмы сортировались по рейтингу - от меньшего рейтинга к большему.

В методе main() создайте список из нескольких фильмов: List<Movie> movies = new ArrayList<>();

movies.add(new Movie("Интерстеллар", 8.7));

movies.add(new Movie("Шрек", 8.1));

movies.add(new Movie("Начало", 8.8));

movies.add(new Movie("Веном", 6.6));

Выведите список фильмов до сортировки.

Отсортируйте список с помощью собственного компаратора: movies.sort(new MovieRatingComparator());

Выведите список после сортировки.



Задание 14.2 Утилита и тесты

Создайте класс XMLUtils, в нем создайте метод:

public static String createEmptyElement(String tagName) Метод принимает название XML-тега и возвращает строку с открывающим и закрывающим тегом. В случае если в метод передан null или пустая строка ("") - возвращать значение "<invalid/>"

Пример: createEmptyElement("user"); //вызов метода // возвращаемый результат: <user></user>

Разработать 2 или более тестов на этот метод. Для написания тестов примените паттерна AAA: 1 Arrange(опционально) - 2 Act - 3 Assert

В процессе написания ассертов - обязательно передавать в метод ассерта параметр-сообщение, в котором указывать детали произошедшей ошибки.



Задание 14.3 (ОПЦИОНАЛЬНОЕ, НЕ ОБЯЗАТЕЛЬНОЕ) Рентген для объектов

Создайте произвольный класс, у класса должно быть:

2 или более полей

2 или более методов

2 или более конструкторов

При объявлении полей, методов и конструкторов нужно применить все 4 модификатора доступа хотя бы по одному разу (например - приватное поле, дефолтный метод, публичный конструктор, протектед конструктор).

Сделать сканер-утилиту - которой на вход передается объект - и в консоль выводится вся информация о нем - все имеющиеся у объекта:

- поля

- методы

- конструкторы

Попробуйте поиграть с кодом сканера-утилиты - заменяя методы: getMethods на getDeclaredMethods, getFields на getDeclaredFields, getConstructors на getDeclaredConstructors.

Вывод о разнице в работе методов вида get* / getDeclared* записать в файле ANSWER.md в пакете с домашним заданием.
